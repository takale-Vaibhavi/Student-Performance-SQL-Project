# Student Performance Analysis using PostgreSQL

## Project Overview

This project analyzes student performance using SQL.

The goal is to identify factors affecting final exam scores such as:

- Study Time
- Previous Failures
- Internet Access
- Family Support
- Social Activity
- Alcohol Consumption

---

## Dataset

Student Performance Dataset

Records: 649 Students

---

## SQL Skills Used

- CREATE TABLE
- ALTER TABLE
- SELECT
- COUNT
- AVG
- GROUP BY
- ORDER BY
- LIMIT
- Subqueries
- Window Functions
- ROW_NUMBER()

---

## Business Questions

1. Does study time improve grades?
2. Which age group performs best?
3. How is the second topper?
4. How is the topper?
5. How many students are in the dataset?
6. Rank students from highest marks.
7. Which gender performs better?
8. Who are the highest-performing students?
---

## Key Insights

- Students with higher study time generally achieved higher scores.
- Students with previous failures scored lower on average.
- Internet access showed a positive impact on performance.
- High alcohol consumption was associated with lower grades.
- Female students slightly outperformed male students.
